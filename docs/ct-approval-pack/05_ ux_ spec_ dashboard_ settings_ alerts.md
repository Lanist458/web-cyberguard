# Web CyberGuard — UX Specification (Settings, Dashboard, Alerts)

## 1. Entry Point (Opt-in)
- Toggle: “Enable Web CyberGuard Protection” (off by default)

## 2. Settings Panel
- Profile
- 2FA (SMS + biometrics)
- Dark/Light Mode
- Donations
- T&C / Privacy Policy / Support

## 3. Dashboard Panel
### Monitoring & Notification panel
- show last events
- user selects alert sensitivity
- strict mode toggle (optional)

### Manual Scan
- Quick Scan (current page/DApp)
- Full Scan (history/caches within allowed scope)
Actions:
- Quarantine
- Delete/Remove

### Virus/Threat DB Update Schedule
- Weekly / Monthly / Quarterly / Semi-annually / Yearly / Automatic

## 4. Alerts
- Block / Warn / Info
- Must include:
  - what was detected
  - why it matters (tags)
  - recommended actions
  - report button
